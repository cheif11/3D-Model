I used a varity of tools in blender, including transform tools, create tools and the material tool. There are so many tools to learn in blender and it would take me months to learn them all so i stuck to the basics while watching a few videos on youtube.

The transform tools include, translate which allows you to move the object by moving the mouse, rotate which allows to turn and rotate objects and scale which lets you resize objects.
The create tool lets you choose between a range of different meshes and curves. It also gives you the option to add text.
The material tool lets you change the colour of your meshes, by clicking on diffuse you can choose between RBG, HSV and hex colour options.
